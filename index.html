Understanding the Date Object

Calendar

The Date object represents a single point in time, including the date and time.

Time

It provides access to various date and time components, such as year, month, day, hour, minute, and second.

Time Zones

Dates can be represented in different time zones, allowing for global application support.

Timestamps

Dates are stored internally as a timestamp, the number of milliseconds since January 1, 1970.

Constructing Date Objects

New Date()

Create a Date object representing the current date and time.

New Date(value)

Create a Date object from a timestamp or date string.

New Date(year, month, day, hour, minute, second, millisecond)

Create a Date object with specific date and time components.
